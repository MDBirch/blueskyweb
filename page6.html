Selecting and Using IT Systems and Software

Chosing the correct systems and software can be critical to a successful project. The choice must be tailored to each project individually. For example, in terms of hardware, if your project is individual, a desktop system may be suitable, but if you need to work with a team, be flexible and be able to work in different locations, or give a presentation, a laptop or portable device would be essential. A smart phone or tablet device may be useful to record ideas or work on the move when a laptop is not suitable.
Software can also help solve problems in a project. If workloads are spread between many individuals, in different locations, video conferencing on web applications such Zoom, Skype and GoToMeeting, can bring organisation and collaboration to a project.
For designing and making a website a range of software and online resources should be employed. Visual Studio code is a software interface that can be used to code in a variety of programming languages, including HTML and CSS used for building websites. VS Code was used to create this website, and i found the interface is well laid out and easy to use. Trello boards are also user friendly and recommended for planning any project.
