Analysis on how Technologies have Helped Achieved Outcomes


(1.6)


Resources such as Trello and Github have been valuable tools to help achieve the outcomes of the project. Trello's visual planning style really suited me, and by adding to the tasks and updating the status of the seperate stages of the process kept the whole project on course. Click this link to view my trello board used for this project.

Github is a code hosting platform. You can upload code, review it and check out other people's projects. Github has been a great way to publish my website quickly and easily. It is a good way to create a profile and have multiple projects there, for others to look at what you have created. A link to my Github repository is here.
