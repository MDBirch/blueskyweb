Access Issues and Disability


(1.6)


It is important that all people can access the website, so disability issues need to be identified, and solutions implemented to help these users. Sight issues are probably the most common problem when using a website. Font size and style was considered in this regard when designing the layout of pages to make sure the text was readable, and not too small. Also button sizes were not to be too small and, clearly able to click on. Also users with sight problems may use a lower resolution of display to make text appear larger, so the website was tested on different window sizings to make sure the pages displayed properly. Also the website was tested in night mode and blue light reduction settings on devices, to make sure text was readable with these settings turned on.
